# Doubleword

`batchling` is compatible with Doubleword through any [supported framework](../frameworks.md)

The following endpoints are made batch-compatible by Doubleword:

- `/v1/chat/completions`
- `/v1/embeddings`
- `/v1/moderations`
- `/v1/completions`
